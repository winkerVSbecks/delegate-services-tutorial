A delegate service allows you to access and control a directive. It also allows you to have multiple instances of a component in the same view.

This is a demo of how the Ionic Framework uses delegate services for their directives.

The nav-bar directive can be controlled using the $navBarDelegate service.

<nav-bar
  delegate-handle="my-navbar"
  title="My Awesome App">
</nav-bar>

The delegate service allows us to grab individual instances using the delegate-handle.

$navBarDelegate
  .$getByHandle('my-navbar')
    .doSomething();
