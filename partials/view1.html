We can use $navBarDelegate to call method such as the change of title, picking the direction of animation and showing/hiding the back button.

Alternatively we can get data from the directive, such as the navigation history.

How Does It Work?

navbar-delegate-service lists all the methods available through the public API. These methods are actually defined in the navbar-controller. The delegateService is a function which maps these methods on to the service, when a handle is registered.
